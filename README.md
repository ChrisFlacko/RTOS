# RTOS
This project delivers an own RTOS specifically for STM32 Nucleo boards

First aim: Basic RTOS

1) OS
- This file is connecting everything and has functions like Init, Start and Delete

2) Task
- Definition/Creation of tasks and TCB

3) Scheduler
- Scheduler: Add new task, select new task usw. 

4) Port 
- Cortex specific: PendSV, SysTick, Initial Stack, Start first task
